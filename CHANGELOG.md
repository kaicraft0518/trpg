# TRPG Table v1.0.0 リリースノート

**リリース日:** 2026-03-12
**対象システム:** クトゥルフ神話TRPG オンラインセッション支援Webアプリ

---

## 初回リリース

TRPG Table は、クトゥルフ神話TRPGのオンラインセッションをブラウザだけで行えるように設計された支援ツールです。
GitHub Pages 上で公開され、Firebase Realtime Database を利用してリアルタイム通信を行います。
インストールは不要で、PC・スマートフォンなど様々なデバイスから参加できます。

---

## 主な機能

### 認証・アカウント

* ユーザーID（ハンドル）とパスワードによるアカウント登録・ログイン
* Googleアカウントによるログイン
* 既存アカウントへのGoogleアカウント連携
* プロフィール画像・表示名の変更
* テーマ設定（ライトモード / ダークモード）の保存

### ホーム

* セッション作成（招待コード自動生成）
* セッション履歴の表示および再入室
* セッション参加時のキャラクター選択
* 招待通知の受信および承認 / 拒否

### キャラクターシート

* 「いあきゃら」からエクスポートしたキャラクターデータのインポート
* HP / SAN / MP のリアルタイム表示および増減操作
* 能力値・技能一覧の表示（カテゴリ折りたたみ対応）
* 参加者キャラクターの閲覧
  （プレイヤーは閲覧のみ、KPは編集可能）
* KP設定によるプレイヤーのキャラクター切り替え許可 / 禁止

### ダイス機能

* D4 / D6 / D8 / D10 / D12 / D20 / D100 に対応
* 技能ロール（技能値入力とクリティカル・成功・失敗・ファンブルの自動判定）
* SANチェック（SAN値に基づく成功判定とSAN減少処理）
* ロール履歴の表示
* KP専用シークレットロール（結果はKPのみ表示）

### チャット・ログ

* マルチチャンネルチャット
  （メインチャット / 情報 / 雑談）
* 個人メモチャンネル（本人のみ閲覧可能）
* 接続・切断やHP/SAN/MP変化を記録するログチャンネル
* KPによるチャンネル作成および管理
* 未読メッセージ表示（モバイルナビゲーション対応）

### 参加者・セッション管理

* 参加者のオンライン状態をリアルタイム表示
* 参加者サイドバーにHP / SAN / MPバーを表示
* 一括ダメージ適用（KP専用：複数キャラクターにHP/SAN/MPをまとめて増減）
* 戦闘順序（イニシアティブ）管理およびNPC追加
* セッションタイムライン記録
* 手掛かり管理
* シーン設定（タグおよび描写テキスト）
* KP設定
  （セッション名変更、プレイヤーロール管理、チャンネル管理、セッション設定）

### モバイル対応

* モバイル向け下部ナビゲーション
* 参加者タブの全画面表示
* タッチ操作に最適化したインターフェース

---

## 既知の制限

* キャラクターのインポートは「いあきゃら」形式のみ対応
* NPCへのSAN / MPの一括適用は未対応（HPのみ対応）
* オフライン状態での操作はサポート対象外
