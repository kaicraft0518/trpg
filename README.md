# TRPG Table

TRPG Table は、ブラウザ上で動作するTRPGセッション支援ツールです。

インストール不要で、ダイスロールやキャラクター管理などを簡単に行うことができます。

**Web Site:**
https://kaicraft0518.github.io/trpg

---

## Features

**ダイスロール**
  任意のダイスロールを簡単に実行できます。

**キャラクター管理**
  キャラクターデータを保存・管理できます。

**チャット機能**
  セッション中のコミュニケーションに使用できます。

**データ共有**
  セッション参加者とデータを共有できます。

---

## Target TRPG

特定のTRPGシステムに依存しない **汎用TRPGツール** です。
主に **Call of Cthulhu (CoC)** のセッションでの利用を想定しています。

---

## Usage

1. サイトにアクセス
2. 使用したい機能を選択
3. セッション中にツールとして利用

インストールや特別な設定は必要ありません。
